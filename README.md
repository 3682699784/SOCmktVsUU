# Python电商用户行为分析系统简介

## 核心功能
- 用户管理
- 登录注册
- 商品信息管理
- 商品详情
- 加入购物车
- 下单
- 评论
- 公告资讯
- 电商用户行为数据可视化大屏
- 商品信息推荐

## 适用场景
- 电商网站数据分析
- 用户行为追踪
- 商品推荐优化
- 营销策略制定

## 技术特点
- 采用Django框架进行后端开发
- 使用MySQL数据库存储数据
- 前端使用Vue.js实现用户界面
- 提供数据可视化大屏，直观展示用户行为数据
- 支持远程调试，确保系统稳定运行
